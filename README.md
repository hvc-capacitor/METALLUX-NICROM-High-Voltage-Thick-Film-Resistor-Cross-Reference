## 📌 Project Overview

**METALLUX** (Italy) and **NICROM** manufacture high voltage thick film resistors used in precision applications:

- Medical imaging (CT/X-Ray voltage dividers)
- High voltage power supplies
- Test & measurement equipment
- Industrial control systems
- Scientific instruments

This repository provides **verified cross-reference data** for replacing METALLUX/NICROM resistors with **HVC HVR/GHP series alternatives**.
